# EmailInput

I am an input field for an email address.

<Playground>
  {() => {
    const [value, setValue] = useState("editme@example.com");
    return <EmailInput value={value} onChange={setValue} />;

</Playground>

I can have a placeholder text.

<Playground>
  {() => {
    const [value, setValue] = useState("");

      <EmailInput

        value={value}
        onChange={setValue}
      />
    );
  }}
</Playground>
